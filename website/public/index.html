COMMON QUESTIONS

Everything users need to know.

Do I enter my Device ID on the website?

No. The website only verifies Discord and issues a trial key. The unchanged MatchIntel app sends its normal Device ID during activation, and the backend performs the device check there.

Can someone use a second Discord account for another trial?

They may be able to claim another key with another Discord account, but the backend rejects that key when it is activated on a device that has already used a MatchIntel website trial.

Does the website store raw Device IDs?

No. The backend stores only a stable cryptographic hash created with your existing private DEVICE_HASH_PEPPER variable.

How does lifetime purchasing work?

The Buy Lifetime buttons use the LIFETIME_BUY_URL Railway variable, so you can point every button to Stripe, Sellix, Lemon Squeezy or another checkout page without editing code.

What Discord permissions are requested?

Only the OAuth identify scope is requested. It provides the stable Discord user ID and basic profile needed to enforce one website trial per account.
