PRIVACY

Privacy overview

MatchIntel uses Discord OAuth to verify free-trial claims. The website requests the Discord identify scope, which provides a stable Discord user ID and basic profile information.

Trial protection

The website does not ask for a Device ID. When a trial key is activated inside MatchIntel, the licensing backend receives the Device ID through the app's existing activation request and stores only a cryptographic, peppered hash. The raw Device ID is not stored in the free-trial usage table.

Stored information

  • Discord user ID and display name for trial and license management
  • License records, status, expiration and device limits
  • Hashed device identifiers for licensing and trial-abuse protection
  • Operational audit and security records

Third parties

Discord processes OAuth sign-in. Railway hosts the website, backend and database according to the configuration selected by the MatchIntel operator.

Contact

Use the support options on the MatchIntel website for privacy questions.
